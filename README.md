# Frontend & Dashboard E-Commerce

This repository contains both the client-facing frontend and the administrative dashboard for the e-commerce application.

## Project Structure

- **[Exclusive-e-commerce](./Exclusive-e-commerce)**: The customer-facing storefront built with React, Vite, and modern CSS/styling.
- **[ecommerce-dashboard](./ecommerce-dashboard)**: The admin dashboard for managing products, categories, orders, and users.

## Backend Service

The backend code is hosted and deployed separately on **Render**.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installing Dependencies

To install dependencies for both projects:

```bash
# Install storefront dependencies
cd Exclusive-e-commerce
npm install

# Run dashboard dependencies
cd ../ecommerce-dashboard
npm install
```

### Running Locally

To run the projects locally in development mode:

```bash
# Run storefront
cd Exclusive-e-commerce
npm run dev

# Run dashboard
cd ecommerce-dashboard
npm run dev
```
